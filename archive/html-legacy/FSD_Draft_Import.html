4. FUNCTIONAL ASPECT (Update)

4.1 Menu BTP (Bahan Tambahan Pangan)

Menu ini digunakan untuk mengelola data master Bahan Tambahan Pangan.

4.1.1 Layout Index BTP

Fungsi Utama:

  • Menampilkan daftar semua BTP dalam tabel dengan DataTables.
  • Feature Search global untuk mencari BTP berdasarkan code atau name.
  • Filter by status (Active/Inactive).
  • Pagination (25 rows/page) dan Sorting per kolom.

Aksi & Navigasi:

Button/Action Navigasi Keterangan
Add New BTPDetail.html (Mode Create) Membuka form kosong untuk create BTP baru.
Edit BTPDetail.html (Mode Edit) Membuka form terisi data untuk melakukan perubahan data BTP.
Delete - Melakukan Soft delete (set bitActive = 0) dengan konfirmasi.

4.1.2 Layout Detail BTP

Fungsi Utama:

  • Form input untuk mendefinisikan Kode dan Nama BTP.
  • Validasi keunikan kode BTP.

Keterangan Field:

Field Name Mandatory Validasi Keterangan
BTP Code Yes Unique, No Space, Alphanumeric Kode unik BTP.
BTP Name Yes Min 3 Char Nama deskriptif BTP.
Active No - Status aktif data.

4.6 Menu Test Parameter Conversion

Menu ini digunakan untuk mengatur konversi parameter uji dan target per sub group.

4.6.1 Layout Index Test Parameter

Fungsi Utama:

  • List konfigurasi konversi parameter.
  • Menampilkan rumus konversi secara ringkas.

Aksi & Navigasi:

Button/Action Navigasi Keterangan
Add New TestParameterConversionDetail.html Tambah konfigurasi konversi baru.
Edit TestParameterConversionDetail.html Ubah konfigurasi.
Set Target TargetPerRMCategoryManage.html Mengatur target spesifik per RM Sub Group.

4.6.2 Layout Detail Test Parameter

Fungsi Utama:

  • Menentukan rumus konversi: (Nilai - Factor) / FactorNumber.
  • Menandai apakah parameter berlaku untuk cairan (Liquid).

Keterangan Field:

Field Name Mandatory Validasi Keterangan
Test Code Yes Unique Kode Test.
Convert Parameter Yes Selection Parameter asal (dari Master Parameter).
Factor No Numeric Nilai pengurang dalam rumus.
Factor Number No Numeric Nilai pembagi dalam rumus.
Convert UOM No - Satuan hasil konversi.
Liquid No Checkbox Penanda jika berlaku untuk bahan cair.
