# Deliverables FSD (terbaru, tanpa timestamp)

Folder ini menyimpan **satu salinan terbaru** per dokumen FSD yang di-build dari modul Engine.
File ditimpa setiap build — **jangan** simpan arsip ber-timestamp di sini (hemat storage Git).

Arsip ber-timestamp ada di:

`D:\Work\Documentation\SHP\Project Log\{TAHUN}\{NNN}. {NAMA_PROYEK}\`

Konfigurasi: `lib/fsd_deliver.py` + `DeliverableConfig` di `ModuleBuildConfig`.
